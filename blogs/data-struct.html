1 Skip list

在有序链表的基础上加多层索引，用于提高查询新能。 

|1|          ->      |5|
 |                    |
|1|    ->     |3| -> |5|    ->     |7|
 |             |      |             |
|1| -> |2| -> |3| -> |5| -> |6| -> |7| -> |8|

2 hyperloglog

基数计数（cardinality counting）算法，用于统计集合中不重复元素的个数。hyperloglog是概率算法，不存储数据集合本身（内存使用少），通过概率统计方法预估基数值（有误差）。（怀疑适用范围，例如：领导对用户活跃数的统计误差几乎是零容忍的）
